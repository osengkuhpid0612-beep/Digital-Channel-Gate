# 數位渠道門 - Digital Channel Gate

竹泊爾王國精靈專用的真善忍淨化資訊中繼站

## 架構

```
外部網際網路 (時事/新聞/科技)
        ↓
GitHub Actions / 中繼站 API (光速拉取 raw 數據)
        ↓
小蜥蜴真善忍檢測引擎
  • 審查「真」：過濾虛假報導、標題黨與捏造資訊
  • 審查「善」：過濾惡意攻擊、仇恨與煽動言論
  • 審查「忍」：過濾極端偏激、急躁無稽之談
        ↓ (淨化後的純淨知識晶體)
竹泊爾王都 (26 位精靈集體推演)
```

## 資料夾結構

- `data/raw/` - 原始爬取數據
- `data/filtered/` - 淨化後數據
- `data/metadata/` - 元數據與時間戳
- `config/` - 設定檔
- `scripts/` - 爬取與驗證腳本

## 工作流

1. GitHub Actions 每日執行爬蟲
2. 收集各類網路資訊
3. 小蜥蜴進行真善忍審核
4. 合格資料存入 `data/filtered/`
5. 通訊器讀取淨化後的資料

## 資訊來源

- 新聞：BBC News、Reuters、AP News
- 科技：TechCrunch、Hacker News、GitHub Trending
- 學術：ArXiv 論文

## 真善忍檢測規則

### 「真」- 虛假信息過濾
- 檢查來源可信度
- 驗證事實聲明
- 識別標題黨與誤導
- 檢測深偽與造假內容

### 「善」- 惡意內容過濾
- 檢測仇恨言論
- 識別煽動性內容
- 檢測人身攻擊
- 過濾暴力與極端內容

### 「忍」- 極端偏激過濾
- 識別情緒化的極端觀點
- 檢測陰謀論
- 過濾非理性的急躁言論
- 識別缺乏證據的激進主張

## 使用方式

通訊器透過 GitHub API 定期拉取 `data/filtered/` 中的資料，注入給 26 位精靈使用。
